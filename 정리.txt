Http 프로토콜
Get , Post 방식이란?
: 클라이언트에서 서버에게 데이터를 보내는 방식
1) Get 방식
 : 서버에게 보내는 데이터를 URL에 append 시켜서 보내는 방식 (size제한이 있음)
 : 서버상의 데이터를 조회할 때 사용함
 : url?key1=value1&key2=value2&key3=value3
https://search.naver.com/search.naver?sm=top_hty&fbm=1&ie=utf8&query=%ED%8C%8C%EC%9D%B4%EC%8D%AC
2) Post 방식
 : 서버에 보내는 데이터를 request 스트림의 Body에 append 시켜서 보내는 방식(size에 제한이 없음)
 : 서버상의 데이터를 변경할 때 사용하거나, 파일업로드와 같은 요청일때 
 : 중요한 데이터(주민번호,카드번호)를 서버로 보낼때 사용하는 방식
